# 关于

本项目是一个用于自动登录校园网用户自助服务系统的工具，旨在帮助用户便捷地获取个人信息并管理校园网连接。

## 技术架构

- **主体语言**：Python
- **登录服务组件**：C++（用于开机最高优先级执行登录）
- **验证码识别**：基于 PyTorch 训练的模型

## 主要功能

- 全自动登录校园网用户自助服务系统
- 自动识别并处理验证码
- 获取用户个人信息
- 提供图形化操作界面

## 关于开源说明

本项目原本考虑在 3.0 版本开源，但由于我们学校校园网的退出登录逻辑存在设计缺陷，开源后相关代码可能被恶意利用。为保障校园网络安全，我们决定暂不开源项目代码。

## 注意事项

本项目仅限本校学生用于个人学习和便捷登录目的，请勿用于任何破坏校园网络安全或违反校规的行为。
